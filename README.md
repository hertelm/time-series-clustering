# time-series-clustering

The two datasets used in the publication "[A comparison of local, cluster-specific and global Transformer models for forecasting electrical loads of individual buildings and substations](https://doi.org/10.1109/PowerTech59965.2025.11180482)" can be downloaded from this URL: [https://bwsyncandshare.kit.edu/s/bWA6QaLZkgpp2Ak](https://bwsyncandshare.kit.edu/s/bWA6QaLZkgpp2Ak)

The original sources are:
- For the Electricity Load Diagrams dataset: [https://archive.ics.uci.edu/dataset/321/electricityloaddiagrams20112014](https://archive.ics.uci.edu/dataset/321/electricityloaddiagrams20112014), with a [CC BY 4.0](https://creativecommons.org/licenses/by/4.0/legalcode) license.
- For the Ausgrid solar home dataset: [https://www.ausgrid.com.au/Industry/Our-Research/Data-to-share/Solar-home-electricity-data](https://www.ausgrid.com.au/Industry/Our-Research/Data-to-share/Solar-home-electricity-data), published by Ausgrid. The data is no longer available at the original source. According to [https://researchdata.edu.au/solar-home-electricity-data/3823405](https://researchdata.edu.au/solar-home-electricity-data/3823405), it has a [CC BY](http://www.opendefinition.org/licenses/cc-by) license.
